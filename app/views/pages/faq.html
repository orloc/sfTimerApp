FAQ / Guide

What does this do ?

You can create "timer groups" those timer groups can be thought of as buckets of timers. Each group can be shared with specific individuals based on the owner(s) choosing. At the moment there is only one type of permission that can be granted

  • Authorization / Login / Registration
  • Management of federated timer groups
  • Creation of timers inside timer groups
  • Real time updates for users looking at the same board.
  • Sharing of timer groups ( no distinction between roles at this time - all are owners)

How do I use this application ?

Upon registration / login you will see something similar to the image below. The first thing you will want to do is create a new timer group found via the form in the bottom left hand corner.

The leftmost picture below has three section markers which will be used in later parts of the guide.

Account / Invitation Management ( top right )

Menu (top left)

Controls ( bottom left)

Once you have added a group you will see it appear underneath the MENU. Click on this group to select it. You will see the screen / controls change to a new section.

Buttons on the controls section are split into TWO GROUPS - "Timers" and "Group" control sections.

  • Timer Section
    • Adds a new timer
  • Group Section
    • Adds a new group
    • Manage permissions for the selected group
    • Edit's currently selected group's name / description
    • Deletes the currently selected group and all associated timers / initiations

What does not do?

Parse your client logs or interact with your physical machine / game.

What is the intent of this project?

To provide a integrated easy to use place to share timers and aggregate mob tracking efforts.
